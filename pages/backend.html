1. Timestamp Microservice

The Timestamp Microservice challenge on freeCodeCamp is a beginner-friendly project that helps developers practice building simple backend applications using JavaScript.

Challenge Overview:

  • Build a backend API service that receives a date string as an input and returns a JSON object containing the following information: Unix timestamp for the given date.
  • UTC date and time string formatted as "YYYY-MM-DDTHH:MM:SS.000Z".
  • Date and time string formatted according to the user's local time zone.
  • Handle invalid date inputs gracefully and return appropriate error messages.
  • Use Express.js for routing and handling requests.
  • Utilize the Date object in JavaScript to manipulate dates and timestamps.
  • Implement unit tests to ensure the functionality of the API.

Skills learned:

  • Building basic backend APIs with Express.js.
  • Working with dates and timestamps in JavaScript.
  • Designing and implementing unit tests for JavaScript code.
  • Understanding HTTP requests, responses, and JSON data format.

2. Request Header Parser

The Request Header Parser challenge on FreeCodeCamp is a beginner-friendly project aimed at helping developers practice building backend applications with JavaScript.

Challenge Overview:

  • Build a backend API service that accepts an HTTP request.
  • Extract and parse the X-Forwarded-For header from the request.
  • Extract the user's IP address from the parsed header value.
  • Identify the user's language based on the Accept-Language header.
  • Determine the user's operating system based on the User-Agent header.
  • Return a JSON object containing the extracted information: ipaddress: User's IP address.
  • language: User's preferred language. software: User's operating system.

Skills learned:

  • Building basic backend APIs with Express.js.
  • Parsing and manipulating HTTP headers.
  • Identifying user information from request headers.
  • Working with JSON data format.

3. URL Shortener Microservice

The URL Shortener Microservice challenge on FreeCodeCamp provides an excellent opportunity for developers to practice building backend APIs that handle URL shortening functionalities.

Challenge Overview:

  • Build a backend API service that accepts a long URL as an input parameter.
  • Generate a unique and short identifier for the long URL.
  • Store the mapping between the long URL and the short URL in a database.
  • Allow users to access the original long URL by visiting the short URL.
  • Handle invalid URLs and gracefully return appropriate error messages.
  • Implement unit tests to ensure the functionality of the API.

Skills learned:

  • Building basic backend APIs with Express.js.
  • Working with databases to store and retrieve data.
  • Implementing hashing algorithms for unique identifier generation.
  • Understanding routing and URL handling in Express.js.
  • Writing unit tests for backend APIs with frameworks like Mocha and Chai.

5. File Metadata Microservice

The File Metadata Microservice challenge on freeCodeCamp provides a practical exercise for developers to learn how to build backend APIs that process and extract information from uploaded files.

Challenge Overview:

  • Develop a backend API service that accepts file uploads.
  • Parse the uploaded file and extract metadata like name, size, and type.
  • Support various file types like text, images, and documents.
  • Handle errors like invalid file types or exceeding a size limit.
  • Return the extracted file metadata in JSON format.
  • Implement unit tests to ensure the functionality of the API.

Skills learned:

  • Building basic backend APIs with Express.js.
  • Handling file uploads with libraries like multer.
  • Parsing and analyzing file content based on its type.
  • Returning data in JSON format for integration with other applications.
  • Writing unit tests for API functionalities.
